# CRM Project

This is a Customer Relationship Management (CRM) system developed using ColdFusion and MySQL.

## 🔹 Features

- User Authentication (Login & Logout)
- Session Management
- Customer Management (CRUD operations)
- Request Management
- Activity Logging (Database + cflog)
- Pagination and Search
- Admin Controls
- Export Data (PDF)

## 🔹 Technologies Used

- ColdFusion (CFML)
- MySQL
- HTML, CSS, JavaScript

## 🔹 Modules

### 1. Authentication
- User login and logout
- Session-based access control

### 2. Customers
- Add, view, update, delete customers
- Search functionality

### 3. Requests
- Submit and manage requests

### 4. Logs
- Track user activities
- Pagination support
- Admin delete logs

## 🔹 Logging

- Database logging for UI display
- `<cflog>` for server-side logging and debugging

## 🔹 Purpose

This project is designed to manage customer data efficiently and track user activities in a secure and structured way.
